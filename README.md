몰려드는 좀비를 잡아 경험치 및 무기 부품을 얻고, 스탯과 무기를 업그레이드 해나가며 오랫동안 생존하십시오. ⚔️<br><br>
- 좀비는 제한된 월드맵에 랜덤으로 스폰됩니다
- 좀비에 접촉할 시 플레이어의 체력이 감소하며, 일정시간마다 보스몹이 소환됩니다
- 보스몹을 처치할 시 해당 스테이지가 클리어되며, 더 강화된 좀비와 보스가 소환됩니다
- 맵에 생성되는 랜덤한 시민을 구출하여, 버프를 받을 수 있습니다
